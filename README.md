## Installation &

1 - clone this repo

```bash
https://github.com/malja1500/news-extension
```

2 - Search this address in Google Chrome

```bash
/chrome://extensions
```

3 - Turn on the Developer Mode option

4 -Click on Load Unpacked and select the extension folder in the address below

```bash
news-extension\public
```

5 - Now you can use this extension
